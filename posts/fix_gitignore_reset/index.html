gitignoreが無視される時の対処法

December 5, 2020 · Jongmin Yun, Neu.

https://stackoverflow.com/a/52621453/7339320

一旦リポジトリの変更事項を全部コミットして、プッシュする。

git rm -r --cached

を実行してGitの情報を全て削除する。

git add .

を実行して全てを追加する。

.gitignoreがちゃんと反映されるようになる。
